# ComfyUI Lumin Upload

Upload your ComfyUI generated images and workflow metadata to Lumin.

## Requirements

- An account on [https://luminhub.io/](https://luminhub.io/) with an API key

## Installation

1. Install the custom node or clone this repository to your ComfyUI `custom_nodes` directory
2. Restart ComfyUI

## Setup

1. Click the "Asset Manager" button in the menu bar
2. Go to the Settings tab and enter your API key
3. Select your organization and save
4. In the Main tab, choose your project
5. Select upload mode (Automatic or Manual)

That's it - Your images will now be uploaded to Lumin
